Machine learning

Machine learning is used in internet search engines, email filters to sort
out spam, websites to make personalised recommendations, banking software to
detect unusual transactions, and lots of apps on our phones such as voice recognition.

Python

Python is a computer programming language often used to build websites and
software, automate tasks, and conduct data analysis. Python is a general purpose
language, meaning it can be used to create a variety of different programs and
isn't specialized for any specific problems.

Git

Git is a DevOps tool used for source code management. It is a free and
open-source version control system used to handle small to very large projects
efficiently. Git is used to tracking changes in the source code, enabling multiple
developers to work together on non-linear development.
